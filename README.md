## Pinned Projects

### [SOLIDWORKS PDM API SDK](https://github.com/BlueByteSystemsInc/SOLIDWORKS-PDM-API-SDK)

An open-source SDK for building robust SOLIDWORKS PDM add-ins. It provides helpers, wrappers, and reusable patterns around the official SOLIDWORKS PDM API to simplify add-in development.

### [PDM Add-In Debugger Console](https://github.com/AmenJlili/Console-debugger-tool-for-SOLIDWORKS-PDM-Professional-addins)

A console-based debugging utility for SOLIDWORKS PDM add-ins. It helps developers test add-in commands, inspect selections, and debug logic without repeatedly attaching to `explorer.exe`.

### [SOLIDWORKS VBA Macros Library](https://SOLIDWORKSVBAMacros.bluebyte.biz)

A collection of ready-to-use SOLIDWORKS VBA macros for common automation tasks, including BOM export, PDF batch processing, model updates, and drawing automation.

### [SOLIDWORKS API Extensions](https://github.com/BlueByteSystemsInc/SOLIDWORKSExtensions)

Reusable C# extension methods for common SOLIDWORKS API operations, including feature traversal, sketch handling, model interrogation, and automation helpers.

### [Power-SM](https://github.com/AmenJlili/power-sm)

A batch utility for updating SOLIDWORKS sheet metal parameters such as bend radius, thickness, and K-factor.
